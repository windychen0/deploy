        本项目是一个基于express打造的轻量化项目打包发布平台

        前端页面为UMD引入的Vue3.2 + ElementPlus

        特色是只需npm依赖项即可一键运行,前后端共用工具类

---
    目前已支持功能:

    1.登录
    2.历史版本纪录 (bug)
    3.查看打包脚本 (变量视图√ 渲染视图×)
    4.执行打包脚本 (错误处理 TODO)
---
    待支持功能:

    1.用户 (增删改查)
    2.打包脚本 (增删改)
    3.变量管理 (增删改查)
    4.git log (查询 ,打包信息联动 ,版本回退)
    5.历史版本回退功能
    6.权限管理
